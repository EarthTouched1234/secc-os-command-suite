# SECC OS Chat Bridge

Use this folder when Codex, voice ChatGPT, or another local session hits context, tool, or permission limits.

## Folder map

- `CURRENT.md`: The live handoff note. Read this first.
- `inbox/`: Messages or instructions from SunNi to Codex.
- `outbox/`: Drafts, summaries, or replies Codex prepares for SunNi.
- `checkpoints/`: Resumable build notes, commands run, decisions, and next steps.
- `archive/`: Old handoffs that should stay available but are no longer active.

## Operating rule

Before a long build, update `CURRENT.md` with the goal, current state, next action, known blockers, and exact files or links that matter.

When a session gets limited, paste or point the next Codex or voice ChatGPT session to `CURRENT.md`.
